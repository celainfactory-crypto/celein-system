## 🎯 نظرة عامة

نظام إداري ومحاسبي ذكي ومتكامل لإدارة المصنع بالكامل (الإنتاج، المخزون، المبيعات، التكاليف). يعمل بالكامل في المتصفح، يحفظ البيانات محلياً، ويعرض كل المؤشرات بشكل لحظي.

## ✨ المميزات الرئيسية

- **لوحة تحكم لحظية** (Dashboard) بـ 8 مؤشرات أداء رئيسية
- **التكلفة الفعلية للكرتون** تُحسب لحظياً من المعادلات
- **نقطة التعادل** (Break-Even) موزونة على كل الأصناف
- **نظام صلاحيات متدرج** (6 أدوار)
- **مخازن**: رصيد = افتتاحي + إنتاج - مصروف
- **مناديب ووكلاء** بمتابعة المديونيات والتحصيل
- **مختبر ومحطة** لتوثيق الاستهلاك والفحوصات
- **تصدير PDF / Excel** لكل التقارير
- **نسخ احتياطية** بصيغة JSON

## 🧮 المعادلات الأساسية

### تكلفة الكرتون الواحد
```
التكلفة = مواد خام + رواتب + مياه + مختبر + صيانة + إهلاك + تالف
التالف = 2% من قيمة المواد الخام
```

### نقطة التعادل
```
نقطة التعادل الشهرية = إجمالي المصروفات العمومية ÷ متوسط هامش المساهمة الموزون
هامش المساهمة = متوسط السعر - التكلفة الفعلية
الموزون = هامش المساهمة × نسبة كل صنف من الإنتاج
```

### المخزون
```
الرصيد المتبقي = الرصيد الافتتاحي + الإنتاج - المصروفات
```
