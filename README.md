
## 🛠️ التقنيات المستخدمة

- **HTML5 / CSS3 / JavaScript (Vanilla)**
- **Chart.js** للرسوم البيانية
- **jsPDF** لتصدير PDF

- **Web App Manifest** للتثبيت كتطبيق

---

## 📱 التثبيت كتطبيق (PWA)

1. افتح https://celain.ink في متصفح الجوال
2. اضغط "إضافة إلى الشاشة الرئيسية" من قائمة المتصفح
3. سيظهر كأيقونة تطبيق باسم "سيلين"

---

## 💾 النسخ الاحتياطي

- من صفحة الإعدادات → تصدير JSON
- يحفظ نسخة كاملة من كل البيانات
- يمكن استعادتها في أي جهاز عبر "استيراد من ملف"
